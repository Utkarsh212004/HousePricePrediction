# HousePricePrediction
A beginner-friendly Machine Learning project that predicts house prices using Random Forest regression, with data preprocessing and an interactive Streamlit interface
# 🏠 House Price Prediction ML Project

## 📌 Overview

This project predicts house prices based on features like size, bedrooms, location, and age using Machine Learning.

## 🚀 Features

* Predicts house prices using trained ML model
* Uses Random Forest Regressor
* Interactive web app using Streamlit

## 🧠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit

## ⚙️ How to Run

### 1. Train Model

```
python train.py
```

### 2. Run Web App

```
streamlit run app.py
```

## 📊 Model Details

* Algorithm: Random Forest Regressor
* Evaluation Metric: R² Score

## 📷 Output

Predicts house price based on user input via web interface.
